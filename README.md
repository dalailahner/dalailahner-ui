# DALAILAHNER-UI

This is a dark UI theme for Atom with a sultry red as main accent color.

Please make sure to also use the corresponding **dalailahner-syntax** theme.

Some inspiration for this theme was Atoms stock dark-one theme, gruvbox and others.


## Preview

![dalailahner-ui and dalailahner-syntax theme in Atom](https://raw.githubusercontent.com/dalailahner/dalailahner-ui/master/img/screenshot1.png)


## Font

You can use any font with this theme you'd like but I prefer [Hack](https://sourcefoundry.org/hack/).
